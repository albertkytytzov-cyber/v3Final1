PERFORM_TEMPLATE_V1 • образец тренировочного дня

Один стандартный день для создания новых шаблонов PERFORM. Строки с типом workout можно связывать с тренировкой устройства. instruction/control остаются деталями тренировки, recovery считается отдельным восстановительным пунктом.

Используйте эту структуру как основу: План → Неделя → День → Сессия → Таблица блоков. Не удаляйте колонку "Тип": она задаёт смысл строки для импорта, аналитики и будущей связи с устройством.

instruction Спуск контроль шага не бежать вниз Это деталь похода, не отдельная тренировка.

Правило образца: одна основная строка workout, детали идут строками instruction/control, восстановление — строкой recovery.
