PERFORM_TEMPLATE_V1 • образец тренировочного дня

Один стандартный день для создания новых шаблонов PERFORM. В таблице оставляем только блок и объём: дополнительные указания записываются общей заметкой к тренировке.

Используйте эту структуру как основу: План → Неделя → День → Сессия → Таблица блоков. Колонка "Контроль" больше не используется: если нужен комментарий, добавляйте его одной общей заметкой под сессией.
